# Car-Rental-System-Java
Console-based Car Rental System developed using Java and OOP concepts.

## Features

- Rent a car
- Return a car
- View available cars
- Customer management
- Rental price calculation

## Technologies Used

- Java
- OOP Concepts
- Collections Framework
- Scanner Class
